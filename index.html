Levenshtein Distance Algorithm

Calculates the minimum number of single-character edits (insertions, deletions, substitutions) needed to change one string into another. The similarity score is computed as:

Score = (maxLength - distance) / maxLength

Example: "Waterlo" → "Waterloo"

Source: W-a-t-e-r-l-o (7 characters)
Standard: W-a-t-e-r-l-o-o (8 characters)
Edit needed: Insert 1 letter "o" → Distance = 1
Score: (8 - 1) / 8 = 7/8 = 87.5%
